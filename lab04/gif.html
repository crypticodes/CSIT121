Behold the GIF!!!

Graphics Interchange Format (GIF): This is primarily used for animated images, but can also be used for still images. There's a maximum of 256 colors, the format uses lossless compression, and it can be interlaced which shows low-res versions until the file fully downloads. I chose this image because it's a fun example of the file type, and reminds me of the Earl of Lemongrab from Adventure Time.
